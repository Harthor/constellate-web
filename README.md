# Constellate Web

Frontend for [Constellate](https://constellate.fyi) — the hidden patterns in what you're already reading.

## What is this

A Next.js app that visualizes constellation data generated by [constellate-engine](https://github.com/Harthor/constellate-engine). Features:

- **Landing page** with featured constellations, how-it-works, and source list
- **Constellation map** — interactive force-directed graph showing constellations as nodes connected by shared ideas
- **5 constellation types** with visual explanations: Chain, Triangulation, Convergence, Absence, Spectrum
- Filters by type, score, and search
- Hub ideas panel (ideas appearing in 3+ constellations)
- Detail panel on click with idea links

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000.

## Data

The app reads from `public/data.json`, which is the output of constellate-engine's pipeline. To update:

```bash
# In the constellate-engine repo:
npx tsx scripts/weekly-run.ts
cp output.json ../constellate-web/public/data.json
```

## Stack

- Next.js 16 (App Router)
- React 19
- Tailwind CSS 4
- react-force-graph-2d (D3 force simulation)
- Static data (no backend required)
